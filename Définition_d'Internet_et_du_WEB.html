Définition d'Internet et du WEB


L’internet, international network en anglais, est l’infrastructure qui permet de faire
transférer des données dans le réseau global. Pour accomplir cette tâche l’internet
utilise deux technologie : le TCP c'est à dire « the transmision central protocole » et l’IP
« internet protocole ». Il existe également le WEB à ne pas confondre avec l’internet.
Le WEB, world wide web en anglais, est une application accessible via l’internet donnant
accès à un ensemble de données. Ainsi l’internet sert de support au WEB.
